Work

ZTC Pathway Mapper

A production-ready web application for tracking Zero Textbook Cost (ZTC) course adoption across academic pathways.

Built to transform CSV data into interactive visual dashboards with comprehensive export capabilities. Features include:

  • Visual Progress Tracking: Color-coded dashboards with real-time statistics
  • Smart Filtering: Filter by term, status, and course search
  • Detailed Analytics: Drill-down into course-level ZTC metrics
  • Professional Exports: Filter-aware PDF, CSV, and PNG reports
  • Zero Cost: 100% client-side, no server required

Technologies: React 18, Tailwind CSS, Chart.js, jsPDF
Status: Production-ready with enhanced error handling
