ZTC Pathway Mapper — End-User Guide

Version: 3.1  ·  Date: June 24, 2026  ·  Prepared by: Bradley Franklin  ·  Audience: ZTC committee, deans, curriculum & IR staff

What this tool does

The ZTC Pathway Mapper shows how close each active degree, certificate, and transfer pathway is to being fully Zero Textbook Cost (ZTC). Upload two institutional CSV exports, click Build dashboard, and explore adoption by term — all processing stays in your browser; data is not uploaded to a server. Open the app: chairmanx.tech/projects/ztc-mapper (recommended). Offline: copy the full ztc-mapper folder including vendor/ — do not share index.html alone.

Two data files required

File What it contains
Program Summary Pathway structure — courses, requirement areas, blocks, and units (Active programs only)
IE ZTC Course Analytics Per-course, per-term ZTC adoption %, survey completion, and section counts (MASTER export)

After the first successful build, your session is saved locally until you choose Load new data.

Quick start — 5 steps

  1. Open the app in Chrome or Edge (Firefox/Safari for spot checks).
  2. Upload the Program Summary CSV — wait for the green Loaded state.
  3. Upload the IE ZTC Course Analytics CSV.
  4. Click Build dashboard (enabled only when both files are loaded).
  5. Use the Term selector in the header (defaults to the latest term with data).

A red error banner usually names a missing column or file read problem — fix the export and try again.

Three views — use the tabs in the top navigation

1

Dashboard

  • Needs attention — pathways below 50% adoption (click to open)
  • Institution overview — six KPI tiles; click to drill into Pathway or Courses (sidebar shows active filter)
  • Compare pathways — distribution, ranking, term trend
  • Export: All pathways summary (CSV) only
2

Pathway

  • Select a pathway in the sidebar (search & type filters)
  • Adoption ring and nested areas (e.g. Cal-GETC AREA 1 → 1A)
  • Click a course row for detail modal + catalog link
  • Export: Current pathway CSV & PDF
3

Courses

  • Search, filter by ZTC status (including CO ZTC), sort by ZTC % / name / survey
  • Detail panel matches the pathway course modal
  • View in course catalog opens Colleague in a new tab
  • Term selector applies to all three views

Course badges

Indicator Meaning
ZTC ≥ 75% of sections zero textbook cost (Chaffey internal bar)
Partial 51–74% of sections ZTC
Not ZTC Below 51%
Purple CO ZTC ≥ one ZTC section (Chancellor's Office) — may appear without green ZTC

Pathway adoption %

Adoption is requirement-based, not a simple course average:

  1. Each scored requirement (area or choice block) must be met with a ZTC course.
  2. In choice blocks, the block counts as ZTC if ≥1 valid option meets the 75% bar.
  3. 100% = every scored requirement can be satisfied with ZTC courses in the selected term.

Courses not offered in the selected term show as not offered; other terms may still appear in trend charts.

Tips & troubleshooting

  1. 1 Build dashboard disabled — both CSVs must show green Loaded before build unlocks.
  2. 2 No pathways after build — confirm Program Summary includes Active programs only.
  3. 3 Course shows “no data” — not in Course Analytics for that term, or not offered.
  4. 4 Wrong term — change Term in the header; all views update together.
  5. 5 Stale session — use Load new data and upload fresh exports.
  6. 6 Storage limit banner — browser storage is full; export if needed, then Load new data or clear site data.
  7. 7 Accessibility — Tab to skip link; Esc closes modals; keyboard works in nav, term select, and export menu.
